# Assets 组件 UI 设计风格指南

面向 Node Lab 底部组件与弹窗的统一规范，确保与 Assets 面板视觉一致。

## 视觉基础
- **色彩**：深色玻璃背景 `#0b0d10`，透明度 92% 左右；前景文字主色 `#ffffff`；次要文字 `#ffffff9a`，说明文字 `#ffffff80`；强调色：蓝色 `#38bdf8`，紫色 `#8b7bff`，绿色 `#2dd4bf`，警告黄 `#fbbf24`。
- **边框与分割**：细边 `1px`，颜色 `rgba(255,255,255,0.12)`；分隔线 `rgba(255,255,255,0.10)`。
- **圆角**：外层容器 `24px`-`28px`；内部卡片 `16px`-`20px`；按钮 `12px`-`16px`；mini pill 使用全圆角。
- **阴影/模糊**：投影 `0 24px 60px rgba(0,0,0,0.55)`；半透明玻璃态可叠加 `backdrop-blur(16px)`。
- **排版**：标题用中粗或粗体；副标题/说明用 12px 左右半透明；全大写分组标签用 10px + 加宽字距。

## 组件形态
- **面板容器**：背景深色+玻璃效果，外边框细线，内边距 16-24px，避免纯黑。
- **列表项/卡片**：卡片式分组，默认边框 `rgba(255,255,255,0.10)`，Hover 提升为 `0.25` 并轻微填充 `rgba(255,255,255,0.05)`。
- **按钮**：扁平+轻描边；主按钮可用实心强调色；图标按钮默认透明背景，Hover 小幅填充 `rgba(255,255,255,0.06)`。
- **标签/筛选**：圆角胶囊，描边淡色；选中态背景轻微填充并加粗文字/改用强调色。
- **输入/文本域**：深底色、细边，聚焦态描边提亮；占位符用说明色。
- **图标块**：方圆角 12-16px 的小底座，淡色填充或描边；保持与 Assets 内 icon tile 一致。

## 布局与间距
- **内边距**：面板上下 16-24px，左右 16-24px；列表项内边距 12-16px。
- **间距**：同级元素 8-12px；列表上下 10-14px；分组之间 16-20px。
- **网格**：弹窗内的图标网格用 2 列，保持卡片宽度一致；宽屏允许 3 列但需保持卡片最小宽度 140-160px。

## 状态与反馈
- **Hover/Active**：描边从 0.10 提升到 0.25，背景叠加 0.04-0.08；主按钮的图标可小幅放大。
- **禁用**：整体降低不透明度到 0.5，禁用 Hover。
- **警告/空态**：用淡黄色描边和图标；空态背景保持深色，文字 60% 透明度。

## 弹窗类型规范
- **导入/导出弹窗**：宽 260-320px，使用与 Assets 相同的容器、分组标签、行内按钮风格；操作项为行按钮+点状强调。
- **模板弹窗**：卡片列表+操作卡；卡片使用描边 hover 提升、主标题白色、日期次要色。
- **新增节点 Palette**：面板+网格卡；卡片有淡底、描边、图标块；文字白色，提示为次要色。
- **放映机（WIP）面板**：大号卡片，头部有强调图标块；正文使用分组卡片描述，保持 Assets 的深色卡+细边。

## 底部胶囊组件
- **Qalam/工具栏/缩放/Assets mini**：高度统一（约 38-44px），圆角 pill，描边 0.10，背景深色玻璃；图标按钮无实底，仅 Hover 浅填充；主操作（运行）用纯色填充图标。
- **MiniMap**：背景与 Assets 一致，描边 0.12，圆角 16px，阴影同 Assets；位置紧贴 mini Assets 上方对齐。

## 动效与过渡
- **出现/隐藏**：使用 150-250ms 的淡入+上移/缩放微动。
- **Hover 过渡**：150ms 以内，ease-out；阴影和描边同步变化。

## 暗色模式细节
- 避免纯黑 #000，使用深灰蓝 #0b0d10；提升对比的同时保持柔和。
- 文本对比：主文字 >= 85% 不透明度；次要 55-65%；禁用 45-50%。

## 代码约定（Tailwind 示例）
- 容器：`rounded-3xl border border-white/12 bg-[#0b0d10]/95 text-white shadow-[0_24px_60px_rgba(0,0,0,0.55)] backdrop-blur-2xl`.
- 卡片：`rounded-2xl border border-white/10 bg-white/2 hover:border-white/25 hover:bg-white/6`.
- 标签：`rounded-full border border-white/15 px-3 py-1 text-[12px] text-white/70 hover:border-white/30`.
- 标题小标签：`text-[10px] font-black uppercase tracking-widest text-white/60`.

## 适配与响应
- 视口变窄时保持比例缩放，优先换行而非缩小字号；保证按钮可触达性（44px 最小可点面积）。
- 大弹窗需在移动端占满宽度，内边距适度收紧，但依旧保持描边与圆角。

## 复用建议
- 封装通用样式常量/类名（如 `panelClass`、`cardItemClass`），所有弹窗与底部组件复用，避免视觉偏差。
